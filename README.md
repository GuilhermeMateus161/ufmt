Face Detection



Este projeto aborda o desenvolvimento de um problema de inteligência artificial (IA) para a
detecção de faces, utilizando três métodos de deep learning para a solução da problemática.

Para o funcionamento do progrma faça a instalaçao das bibliotecas:

cv2(OpenCV)
mediapipe
time
pathlib
numpy
mtcnn
tensorflow

Para instalar as dependencias necessarias para o funcionamento execute o comando pip install -r requirements.txt

Em caso de erros na execuçao utilisando IDE executar via terminal.
